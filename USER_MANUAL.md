# AI Sensei - User Manual (Professor)

## 1. How to create a Knowledge Graph
1. Navigate to the **Architect** module via the main menu.
2. Upload your course syllabus or a relevant textbook (PDF format only).
3. Click **Generate Competency Map**.
4. The system will analyze the document and visualize a graph of concepts and competencies.
5. Tap on any node to view details, including its level in Bloom's Taxonomy.

## 2. How to run the Observer
1. Navigate to the **Observer** module.
2. Click **Start Observation** (or similar button).
3. Ensure your microphone permissions are granted. The system will listen to the classroom discourse.
4. The AI analyzes the audio in real-time (ephemeral processing) to provide metrics like Teacher vs. Student talk time.
5. After the session, use the **Map Insights** feature in the Architect view to mark covered topics in your Knowledge Graph.

## 3. How to Trigger a Crisis (PBL)
1. Open the **Lesson Editor** for a specific Project-Based Learning lesson.
2. Locate the **Crisis Injection** panel.
3. Click the **Inject Crisis** button.
4. This action triggers a critical situation event. Students currently active in the project will see a "System Alert" overlay.
5. Students must complete the recovery protocol tasks to resolve the crisis and continue.

## 4. How to Export Research Data
1. Navigate to the **Analytics** module.
2. In the "Research Engine & Black Box" section, select a **Class** from the dropdown menu.
3. Click **Generate Report** to create a current snapshot of the class performance and behavior.
4. Once generated, use the **JSON Export** or **CSV Export** buttons to download the anonymized dataset for further analysis.
